# KMS User Guide

## Getting Started

### Login

1. Navigate to `https://kms.it-enterprise.solutions`
2. Login with your credentials or use Google/GitHub OAuth

### Interface Overview

The KMS interface consists of:

- **Header** - Search, refresh, user menu
- **Module Navigation** - Switch between modules (IDEAS, DEVELOP, DEPLOY, etc.)
- **Sidebar** - Category tree navigation
- **Main Content** - Project and document views
- **Tools Button** - Access development tools

## Modules

### IDEAS Module
Plan and conceptualize your projects.

**Features:**
- Project specification writing
- AI-powered analysis
- Generate phases, tasks, and guides

### DEVELOP Module
Main development workspace.

**Features:**
- Stage workflow (Analyze → Test → Docs)
- Tabs: Overview, Phases, Tasks, Guides, etc.
- Quick access to Terminal, VS Code, Claude AI

### TASKS Module
Team task management.

**Features:**
- Kanban board view
- Task creation and assignment
- Drag & drop status changes
- Priority management

### DEPLOY Module
Release and deployment management.

**Features:**
- Client management
- Backup creation
- Test and production deployment
- Billing integration

### ANALYTICS Module
Monitor system usage.

**Features:**
- Usage statistics
- Error tracking
- AI usage metrics

### CLIENTS Module
Customer relationship management.

**Features:**
- Client database
- Contact management
- Software catalog
- Billing history

### FINANCE Module
Financial operations.

**Features:**
- Invoice creation
- Payment tracking
- Reminders
- Financial reports

## Tools

### Terminal
Access project terminal directly from browser.

1. Select a project
2. Click Tools → Terminal
3. Terminal opens in project directory

### VS Code
Edit code in web-based VS Code.

1. Select a project
2. Click Tools → VS Code
3. Editor opens with project files

### Claude AI
AI assistant for code review and help.

1. Select a project
2. Click Tools → Claude AI
3. Ask questions about your code

### Git Operations
Manage version control.

- **Status** - View changes
- **Pull** - Update from remote
- **Commit** - Save changes
- **Push** - Upload to remote

### Import Project
Import projects from various sources.

**Sources:**
- Git repository
- Local folder (drag & drop)
- SFTP/SSH
- SMB/CIFS
- NFS

### Local PC Tools
Connect to your local computer.

1. Download client script from Tools → Local PC
2. Run on your computer
3. Access Cursor, Windsurf, Terminal remotely

## Keyboard Shortcuts

- `Ctrl + /` - Toggle sidebar
- `Ctrl + K` - Search
- `Ctrl + S` - Save (in editors)
- `Escape` - Close modal

## Tips

1. **Use WikiSys rules** - Claude AI automatically loads development rules
2. **Regular backups** - System creates daily backups automatically
3. **Module switching** - Use navigation bar for quick access
4. **Project passing** - Use ↑ arrow to pass project between modules

## Troubleshooting

### Can't connect?
- Check internet connection
- Clear browser cache
- Try logging out and back in

### Tools not working?
- Verify project path exists
- Check permissions
- Review browser console for errors

Contact support at dev@it-enterprise.cz
